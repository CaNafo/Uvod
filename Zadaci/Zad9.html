/*Napisati program u programskom jeziku C gdje korisnik treba da unese dvocifren broj, a zatim se na ekranu ispisuje da li je broj djeljiv sa 5, takođe program treba da ispiše odgovarajuću poruku ukoliko nije unijet dvocifren broj */

#include<stdio.h>
main()
{
	int broj;

	printf("Unesite broj: ");
	scanf("%d", &broj);

	if (broj > 9 && broj < 100)
	{
		if (broj % 5 == 0)
		{
			printf("Broj je djeljiv sa 5. \n\n");
		}
		else
		{
			printf("Broj nije djeljiv sa 5. \n\n");

		}
	}
	else
		printf("Broj koji ste unijeli nije dvocifren!\n\n");

	system("pause");
}

Napisati program u programskom jeziku C gdje korisnik treba da unese broj, a zatim se na ekranu ispisuje da li je unijeti broj cio ili realan. */

#include<stdio.h>
main()
{
	float broj;

	printf("Unesite broj: ");
	scanf("%f", &broj);

	if (broj == (int)broj)
		printf("Unešeni broj je cio!\n\n");
	else
		printf("Unešeni broj je realan!\n\n");
	system("pause");
}
Objašnjenje:
U dijelu koda "broj == (int)broj" je primjenjeno eksplicitno kastovanje, tj. konvertovanje jednog tipa podatka u drugi. Postoje dvije vrste kastovanja implicitno i eksplicitno. Prilikom kastovanja treba biti oprezan jer može doći do gubitka podataka. Više o kastovanju možete naći na: Kastovanje.
