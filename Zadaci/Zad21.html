Do While petlja 1

/*Napisati program na C-u kojim se štampaju svi parni brojevi od 0 do 50.*/

#include<stdio.h>
main()
{
	int i=0;
	
	do {
		printf("%d\n", i);
		i = i + 2;
	} while (i < 50);

	system("pause");
}
Razlika između WHILE i DO-WHILE kontrolne strukture: Kod DO-WHILE prvo se izvršava naredba (blok naredbi), a tek potom provjerava uslov. Dakle, kod DO-WHILE kontrolne strukture naredba (blok naredbi) će se izvršiti barem jednom, dok je kod WHILE kontrolne strukture moguće da se naredba (blok naredbi) ne izvrši niti jednom (ukoliko u startu uslov nije bio ispunjen).
